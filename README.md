# ElevatorSaga
Meine Lösung für das Spiel [Elevator Saga](http://play.elevatorsaga.com).

Diese beinhaltet folgende Grundprinzipien:

* Jeder Lift fährt in eine bestimmte Richtung und überprüft bei jedem Stockwerk, ob er anhalten muss.
* Die Mitfahr-Anfragen auf den Stockwerken werden von einem Controller entgegen genommen und aufgrund verschiedener Kriterien einem spezifischen Lift zugewiesen.
